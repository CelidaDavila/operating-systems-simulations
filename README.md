## \# operating-systems-simulations

## 

## Simulations for the Operating Systems course (console-based, C++).
Console-based simulation of operating system process management concepts.

## This repository contains three programs:

* Program 1: Batch Processing Simulation (batches of max 5 processes).
* Program 2: Batch Processing with Multiprogramming (keyboard control: I/E/P/C).
* Program 3: Simulates an Operating System process scheduler using the First Come First Served (FCFS) algorithm.

## Requirements

* C++17 compiler (g++, clang++, or MSVC)
* Console/terminal

## How to Run
Windows (g++ / MinGW)
bash
g++ -std=c++17 main.cpp -o main.exe
main.exe
